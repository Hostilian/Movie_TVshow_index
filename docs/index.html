Sample Queries

Category A - Projection

Show only the title and year columns from all movies

RA: MOVIE[title, year]
SQL: SELECT title, year FROM movie;
Category CN - Left Outer Join

Show all movies with their award information including movies with no awards

RA: MOVIE !< AWARD
SQL: SELECT * FROM movie m LEFT JOIN award a ON m.movie_id = a.movie_id;
Category D2 - Three Table Join

Join movie through movie_actor to actor table

RA: (MOVIE <* MOVIE_ACTOR) <* ACTOR
SQL: SELECT * FROM movie m JOIN movie_actor ma ON m.movie_id = ma.movie_id JOIN actor a ON ma.actor_id = a.actor_id;
Category H1 - Union

Get all unique names from both director and actor tables

RA: DIRECTOR[name] ‚à™ ACTOR[name]
SQL: SELECT name FROM director UNION SELECT name FROM actor;
